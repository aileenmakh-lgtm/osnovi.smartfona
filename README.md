***Сайт "Основы смартфона"***

## Краткое описание проекта

Продукт для Московской Предпрофессиональной олимпиады школьников по профилю "Информамционные технологии" командного кейса №4 "Интерактивная платформа для освоения цифровой грамотности". Предоставляется в виде образовательного веб-сайта для обучения основам работы со смартфонами, предназначенный для пользователей всех возрастов, особенно для тех, кто только начинает осваивать мобильные устройства.
Пользователь может зарегистрироваться и войти в свой аккаунт под уникальным именем и паролем в не менее, чем четыре символа. После этого ему будут доступны уроки сайта двух уровней сложности: начального и повышенного. Чтобы освоить каждый из них, надо просмотреть все уроки и пройти тест выбранного раздела, ответив на все вопросы абсолютно правильно. За прохождение пользователь получит грамоту с его именем, которую можно распечатать. Если что-то неясно по сайту, человек может задать один из готовых удобных вопросов местному чат-боту. Также можно войти в аккаунт администратора («Айлин»), накотором есть доступ к панели управления вместе с созданием видео, редактированием старых и тестов. Все данные пользователя и его успехи сохраняются даже в новых сессиях в браузере. Сайт адаптирован под мобильные устройства.

**Основные возможности:**

- Два уровня обучения: начальный и продвинутый
- Интерактивные уроки с видео-материалами
- Тестирование для проверки знаний
- Система сертификатов (грамот) за успешное прохождение курсов
- Чат-бот помощник с ответами на частые вопросы
- Административная панель для управления контентом
- Полностью адаптивный дизайн для мобильных устройств

**Технологии:** HTML5, CSS3, JavaScript (Vanilla), LocalStorage для хранения данных

Проект учебный, потому в коде используется лишь локальная база данных, по причине чего данные на данный момент не синхронизируются между устройствами и браузерами, а остаются лишь в данном.

1.  **Откройте проект:**
   - Перейдите по ссылке на готовый сайт https://aileenmakh-lgtm.github.io/osnovi.smartfona/
     *ЛИБО* Просто откройте файл `index.html` в любом современном браузере

3. **Доступные тестовые аккаунты:**
   - Администратор: логин `Айлин`, пароль `5555`
   - Обычные пользователи: можно зарегистрировать любого

## Особенности реализации

- Полностью клиентское приложение
- Оптимизация для мобильных устройств: сенсорные цели 44px, предотвращение масштабирования
- Доступность: поддержка клавиатурной навигации, семантическая разметка
- Оффлайн-работа: все функции доступны без интернета после первой загрузки
- Печать сертификатов

## Управление контентом (для администраторов)

Администраторы могут:
1. Редактировать уроки (текст, видео, иконки)
2. Изменять вопросы тестов
3. Добавлять новые уроки и вопросы
4. Загружать видео через iframe-вставку с рутуба

## Браузерная поддержка

Проект работает во всех современных браузерах:
- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+
